# snake

#### 介绍
一个简单的java开发的小游戏-贪吃蛇
画面不太精致，基本都是自带的矩形绘制
主要是在掌握基础技能和一些项目经历


#### 安装教程

java IDE，本人用的是eclipse

#### 使用说明

从Main方法进入，死亡或者Alt+F4退出
